# Triton

项目地址:https://github.com/JonathanSalwan/Triton

## Triton库安装

我习惯使用windows 64位开发环境，因此这里使用vcpkg，命令如下:

```powershell
vcpkg.exe integrate install
vcpkg.exe install triton:x64-windows-static
```
