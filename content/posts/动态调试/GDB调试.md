# GDB调试

| 命令             | 解释                           |
| ---------------- | ------------------------------ |
| attach 进程PID   | 附加进程调试                   |
| next             | 单步步过,简写n                 |
| step             | 单布步入,简写s                 |
| finish           | 结束当前函数，返回到函数调用点 |
|                  |                                |
| print            | 打印地址以及值，简写p          |
| info breakpoints | 查看当前断点情况               |
