## UiAutomator2

项目地址:https://github.com/openatx/uiautomator2

1、安装https://github.com/openatx/weditor，用于查看控件，命令行如下:

```bash
pip3 install -U weditor
```

如果安装遇到错误，在命令行中试试运行命令

```bash
set PYTHONUTF8=1 
```

安装完成后，运行`weditor --shortcut`添加桌面快捷方式。
